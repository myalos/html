matplotlib的安装： pip install matplotlib

matplotlib的导入： import matplotlib.pyplot as plt

简单的画图:

fig, ax = plt.subplots() # 创建一个包含一个axes的figure

ax.plot([1, 2, 3, 4], [1, 4, 2, 3]); # 绘制图像,两个数组元素个数要相同，前面数组是横坐标，后面是纵坐标


四个层级: Figure Axes Axis Tick, 其中Figure对应的是整个显示界面，Axes对应的是一张图，在Axes上画图可以用Axes对象的plot方法，上面代码subplots函数返回的就是一个Figure对象和一个Axes对象

fig, (ax1, ax2) = plt.subplots(1, 2) # 这个是在一个Figure里面画两张图
